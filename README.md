# MBTI 割合診断

16タイプの性格を「割合」で表示するMBTI診断サイト。  
「INFJ 45% / INFP 28%」のように、単純な1タイプ判定ではなく確率分布として結果を可視化します。

## デモ

https://2022311057.github.io/mbti-test/mbti.html

## 特徴

- **割合診断**: Softmax関数とコサイン類似度を用いて16タイプの確率分布を算出
- **レーダーチャート**: 8つの認知機能スコアをビジュアル表示
- **レスポンシブ対応**: PCとスマホで最適化されたレイアウト
- **32問・約5分**: 直感で答えるだけで診断完了

## 使い方

ブラウザでデモURLを開き、32問に答えるだけです。インストール不要。

## アルゴリズム

1. 32問（各5段階）の回答を8つの認知機能スコアに集約
2. 各MBTI16タイプの理想ベクトルとのコサイン類似度を計算
3. Softmax関数（温度パラメータ T=0.10）で確率分布に変換

**8つの認知機能**

| 略称 | 名称 |
|------|------|
| Ne | 外向的直感 |
| Ni | 内向的直感 |
| Se | 外向的感覚 |
| Si | 内向的感覚 |
| Te | 外向的思考 |
| Ti | 内向的思考 |
| Fe | 外向的感情 |
| Fi | 内向的感情 |
