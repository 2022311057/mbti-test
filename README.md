# MBTI 割合診断

16タイプの性格を「2タイプのブレンド」として表示するMBTI診断サイト。  
単純な1タイプ判定や16分割の確率分布ではなく、上位2タイプを「INFJ 64% × ENFP 36%」のように再計算し、組み合わせに「静かな夢想家」のようなオリジナルの名前をつけて表示します。

## デモ

https://2022311057.github.io/mbti-test/mbti.html

## 特徴

- **性格ブレンド診断**: Softmax関数とコサイン類似度で16タイプの確率分布を算出し、上位2タイプを100%基準に再計算してブレンド比率として表示。組み合わせごとに固有の名前（例:「静かな夢想家」）を自動生成
- **レーダーチャート**: 8つの認知機能スコアをビジュアル表示
- **レスポンシブ対応**: PCとスマホで最適化されたレイアウト
- **32問・約5分**: 直感で答えるだけで診断完了（4つの対立ペア×8問。一部は「AよりBの方が合っている」形式の対比型項目で、全項目に同意してしまうバイアスを防ぐ。出題順はセッションごとにシャッフルし連続出題を回避）

## 使い方

ブラウザでデモURLを開き、32問に答えるだけです。インストール不要。

## アルゴリズム

1. 32問（各4段階）の回答を8つの認知機能スコアに集約
2. 各MBTI16タイプの理想ベクトルとのコサイン類似度を計算
3. Softmax関数（温度パラメータ T=0.10）で確率分布に変換
4. 上位2タイプの確率を合計100%として再計算（ブレンド比率）
5. 上位2タイプを4つの気質グループ(NT/NF/SJ/SP)に分類し、ベースタイプの形容詞＋アクセント気質の名詞を組み合わせてブレンド名を自動生成（同気質同士の場合は専用の強化名詞を使用）

**8つの認知機能**

| 略称 | 名称 |
|------|------|
| Ne | 外向的直感 |
| Ni | 内向的直感 |
| Se | 外向的感覚 |
| Si | 内向的感覚 |
| Te | 外向的思考 |
| Ti | 内向的思考 |
| Fe | 外向的感情 |
| Fi | 内向的感情 |
